# Avaliação de LLMs na Detecção Automática de Vulnerabilidades em Código PHP


## Vulnerable PHP Repository LLM

Repositório contendo arquivos PHP com exemplos de vulnerabilidades para fins de avaliação de ferramentas de pentests.

Foi feito um prompt solicitando ao ChatGPT a criação de 10 arquivos de exemplos.
